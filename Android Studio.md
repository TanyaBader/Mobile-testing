### Краткая пошаговая инструкция "для коллеги" по созданию iOS-симулятора в Xcode:

Симулятор - это не отдельная программа, он идёт вместе с Xcode.

1. Установи Xcode

    - Скачай Xcode из Mac App Store и открой его.
    - При первом запуске он сам докачает нужные дополнения (симуляторы, SDK).
    - На что стоит обратить внимание:
        - Если Xcode ни разу не запускался, симуляторы потом могут просто не появиться.

2. Создай новый проект

    - Открой Xcode;
    - Нажми Create a new Xcode project;
    - Выбери iOS - App;
    - Нажми Next;
    - Важно не перепутать:
        - Нужен именно iOS, не macOS и не Game.

3. Заполни настройки проекта

    - Product Name - любое;
    - Interface - SwiftUI (или Storyboard, если сказали использовать его);
    - Language - Swift;
    - Bundle Identifier можно оставить как есть;
    - Нажми Next и Create;
    - Если появится ошибка про Signing - это нормально;
    - Для симулятора Apple ID не нужен, можно не трогать.

4. Выбери симулятор

    - В верхней панели Xcode (рядом с кнопкой запуска) выбери устройство, например:
        - iPhone 14;
        - iPhone 15 Pro;
        - iPhone SE;
    - Обрати внимание:
        - Если написано Any iOS Device - симулятор не выбран и проект не запустится.

5. Запусти приложение

    - Нажми кнопку Run;
    - Xcode соберёт проект и откроет окно iOS Simulator;
    - Нормально, если первый запуск долгий;
    - Симулятор может запускаться 1–2 минуты.

6. Если симуляторов нет

    - Иногда список пустой - значит, они просто не скачаны.
    - Что сделать:
        - Xcode - Settings - Platforms - iOS - Download
    - Или:
        - Window - Devices and Simulators - вкладка Simulators;
        - Нажать + и добавить устройство вручную;
    - Важно: Версия iOS симулятора должна подходить под версию Xcode.

7. Полезные действия в симуляторе

    - ⌘ + H - кнопка Home;
    - ⌘ + Shift + H - список запущенных приложений;
    - Hardware → Rotate - поворот экрана;
    - Device → Erase All Content and Settings - полный сброс;
    - Если что-то работает странно - сброс симулятора часто помогает.

8. Частые ошибки новичков

    - Приложение не запускается - выбран не симулятор;
    - Нет симуляторов - не скачана платформа iOS;

    - Ошибка подписи - игнорируем, если работаем только с симулятором.
